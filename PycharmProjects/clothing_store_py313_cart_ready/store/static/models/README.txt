📦 CARPETA DE MODELOS 3D - WARSTKI®

Esta carpeta es para guardar tus modelos 3D personalizados (.GLB / .GLTF)

INSTRUCCIONES:

1. Descarga modelos .glb de:
   - Sketchfab: https://sketchfab.com/search?q=t-shirt&type=models
   - CGTrader: https://www.cgtrader.com/free-3d-models/clothing
   - Free3D: https://free3d.com/3d-models/clothing

2. Guarda los archivos aquí:
   store/static/models/nombre-modelo.glb

3. Usa en tu código:
   modelPath: '{% static "models/nombre-modelo.glb" %}'

EJEMPLOS DE NOMBRES:
- camiseta-basica.glb
- jersey-running.glb
- sudadera-deportiva.glb
- polo-azul.glb

RECOMENDACIONES:
- Tamaño: < 5 MB
- Polígonos: < 50,000
- Optimizar con: https://gltf.report/
